# Career Advisor Expert System

Hi! This is a **Career Advisor Expert System** built using Python and Streamlit. It helps students figure out which career suits them best based on their interests, skills, and work preferences.  

## Features
- Choose your interests and skills from a list.
- Get a recommended career instantly.
- Covers careers like:
  - Software Engineer
  - Data Analyst
  - Civil Engineer
  - Accountant
  - Graphic Designer
- Easy and interactive interface with Streamlit.

## How to Run
1. Clone the repository:
git clone https://github.com/Ally622/career-advisor-system.git
cd career-advisor-system
2.Install dependencies:
pip install -r requirements.txt
3.Run the app:
streamlit run app.py
Select your facts (skills, preferences, etc.)

The app will recommend a career based on your selections.
